# Polices Personnalisées

Ce dossier contient les fichiers de police pour le site CITY HOTEL KÉTOU.

## Polices Requises

### Back Wild (Titres principaux)
- **Usage** : Titres principaux (h1)
- **Fichiers nécessaires** :
  - `BackWild-Regular.woff2` (recommandé)
  - `BackWild-Regular.woff` (fallback)
  - `BackWild-Regular.ttf` (fallback)

**Où télécharger** : [MyFonts - Back Wild](https://www.myfonts.com/collections/back-wild-graffiti-font-sipanji21)

### Petrov Sans (Titres secondaires)
- **Usage** : Titres secondaires (h2, h3)
- **Fichiers nécessaires** :
  - `PetrovSans-Regular.woff2` (recommandé)
  - `PetrovSans-Regular.woff` (fallback)
  - `PetrovSans-Regular.ttf` (fallback)
  - `PetrovSans-Bold.woff2` (pour les titres en gras)
  - `PetrovSans-Bold.woff` (fallback)
  - `PetrovSans-Bold.ttf` (fallback)

**Où télécharger** : [MyFonts - Petrov Sans](https://www.myfonts.com/collections/petrov-sans-font-fontfabric)

### Montserrat (Texte général)
- **Usage** : Texte général, paragraphes, éléments tertiaires
- **Source** : Google Fonts (chargée automatiquement)
- **Aucun fichier local nécessaire**

## Instructions

1. Téléchargez les fichiers de police Back Wild et Petrov Sans depuis MyFonts
2. Placez les fichiers dans ce dossier (`public/fonts/`)
3. Les polices seront automatiquement chargées par le site

## Format Recommandé

Privilégiez les formats `.woff2` pour de meilleures performances, avec `.woff` et `.ttf` en fallback.

## Note

Assurez-vous d'avoir les licences appropriées pour l'utilisation commerciale de Back Wild et Petrov Sans.
